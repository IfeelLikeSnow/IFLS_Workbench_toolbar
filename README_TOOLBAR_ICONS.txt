IFLS Workbench Toolbar Icons
============================

Icons are installed to: REAPER resource path / Data / toolbar_icons

Files included:
- IFLSWB_explode.png
- IFLSWB_polywav.png
- IFLSWB_slicesmart.png
- IFLSWB_slicemenu.png
- IFLSWB_install.png
- IFLSWB_diag.png
- IFLSWB_cleanup.png

Each icon is 90x30 PNG (3 states: normal / hover / active), per common REAPER toolbar icon convention.
