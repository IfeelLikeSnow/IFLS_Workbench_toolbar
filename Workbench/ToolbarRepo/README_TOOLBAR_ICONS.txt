IFLS_Workbench Toolbar Icons

Where REAPER looks for toolbar icons (so they show up in the icon chooser):
  <REAPER resource path>/Data/toolbar_icons/

This repo includes IFLSWB icons, and also ships a helper installer script:

  Scripts/IFLS_Workbench/Tools/IFLS_Workbench_Install_Toolbar_Icons.lua

Run it if:
- you installed manually (ZIP) and icons don't appear in the chooser,
- or Data/toolbar_icons didn't get populated.
