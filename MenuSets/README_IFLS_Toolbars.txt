IFLS Workbench Toolbars

This folder is the standard REAPER location for toolbar/menu import files (*.ReaperMenu).
Your system-specific toolbar files can be generated by:

Scripts/IFLS_Workbench/IFLS_Workbench_Toolbar_Generate_ReaperMenu.lua

Then in REAPER:
Options -> Customize toolbars... -> (pick toolbar) -> Import... -> select the generated .ReaperMenu
