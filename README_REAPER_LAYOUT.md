# IFLS Workbench Toolbar (REAPER resource layout)

This repo is arranged like REAPER's resource folder:

- Scripts/IFLS_Workbench/...    (ReaScripts: .lua)
- Scripts/IFLS/...              (Utilities)
- Effects/IFLS Workbench/...    (JSFX, do NOT load via ReaScript: Load)
- FXChains/IFLS Workbench/...   (.rfxchain)
- Data/IFLS Workbench/...       (JSON, maps, profiles)
- MenuSets/...                  (.ReaperMenu generated by toolbar generator)

JSFX are discovered by REAPER from the **Effects** folder; they are not registered as Actions.
