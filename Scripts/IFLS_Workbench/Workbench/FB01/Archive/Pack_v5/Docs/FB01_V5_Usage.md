# V5 Usage (FB-01 Voice Macro GUI)

## 1) Requirements
- SWS installed
- ReaImGui installed

## 2) Run
Actions -> Run ReaScript:
- `IFLS_FB01_VoiceMacro_GUI.lua`

## 3) FB-01 settings
- Set **System Channel** on the FB-01 to match the script.
- Choose an **Instrument/Part** (1..8) you want to edit.

## 4) Typical workflow
- Pick Part
- Use Quick Macros (Glass/Metal/Drone/Pad/Bass) as starting points
- Tweak Algorithm/Feedback + TL
- Save to Slot
- Write Track Notes for project recall

## 5) If something doesn't react
- System Channel mismatch
- Editing the wrong Instrument/Part
- MIDI cabling (1824c MIDI OUT -> FB-01 MIDI IN)
