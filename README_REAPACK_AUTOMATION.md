## Import URLs (ReaPack)
Stable:
`https://github.com/<user>/<repo>/raw/<branch>/index.xml`

Nightly:
`https://github.com/<user>/<repo>/raw/<branch>/nightly/index.xml`

## Files
- `.reapack-index.conf` - stable defaults (name/output)
- `index.xml` - stable (generated)
- `nightly/index.xml` - nightly (generated)

## Workflows
- `.github/workflows/reapack-nightly-channel.yml`
  - regenerates `nightly/index.xml` on push to main/master (and commits it)
- `.github/workflows/reapack-stable-release.yml`
  - on tag `v*` generates stable `index.xml`, validates, and uploads as artifact
- `.github/workflows/ci-reapack-validate.yml`
  - PR validation checks stable index is current
