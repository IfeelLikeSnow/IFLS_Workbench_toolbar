## Notes/Highlights (DE)
Kultklassiker („Da Da Da“)
• Rechnen + Spielen
• VL‑Tone-Reihe
ADSR User-Tone (über Taschenrechner-Modus):
• Code 1: Waveform (1=Pulse, 2=Square, 3=Saw, 4=Brass, 5=Piano)
• Code 2: Attack (0–9)
• Code 3: Decay (0–9)
• Code 4: Sustain (0–9)
• Code 5: Release (0–9)
• Code 6: Vibrato (0=aus, 1=ein)
• Code 7: Tremolo (0=aus, 1=ein)
→ Eingabe: [CAL]-Taste, Code eingeben, dann [M+], um Klang zu speichern.

## Besonderheiten/Tech (DE)
Mini-Keyboard/Calculator, monophon, 29 Tasten
